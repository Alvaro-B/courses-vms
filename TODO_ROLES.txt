========
ANACONDA
========
1. Para a actual versión de CentOS 9 precísanse paquetes para poder usar anaconda-navigator dado que ten 
dependencias necesarias para a GUI que non están na máquina.

Comprobado que ao instalar as seguintes dependencias o navigator arranca.
----
sudo dnf install \
    qt5-qtbase \
    qt5-qtbase-gui \
    qt5-qtx11extras \
    xorg-x11-server-Xorg \
    libxkbcommon \
    libxkbcommon-x11 \
    mesa-libGL \
    xcb-util \
    xcb-util-wm \
    xcb-util-keysyms \
    xcb-util-image \
    xcb-util-renderutil \
    mesa-libEGL \
    mesa-dri-drivers \
    pciutils
----
